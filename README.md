# iMPS

A python package for simulating infinite matrix product state (iMPS) evolution including

* Single qubit iMPS evolution
* Two qubit iTEBD
* Local observable expectation values
* Bipartite purities and entanglement entropies
* Local transfer matrix extraction and mutual information

## User guide

Install with `./install.sh` and uninstall with `./unistall.sh`.

The package is poorly documented but you can find example scripts in the `examples` directory.
